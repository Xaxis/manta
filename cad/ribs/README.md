# cad/ribs/

Bistable CFRP tape-spring rib booms. 9 per side, 18 total. Two states: stowed (coiled flat against the spar) and deployed (snapped through to rigid open form).

**Driven by:** `docs/02-structural-budget.md`, `analysis/struct/ribs/`, `analysis/deployment/tape-spring/`.

**Both states must be modeled.** Stowed geometry sets the package thickness budget (<15 cm off body profile, BRIEF). Deployed geometry sets the rib chord, height, and skin attachment line.

To be populated alongside deliverable #2 (mass) and the deployment dynamics work feeding deliverable #5 (symmetry budget).
